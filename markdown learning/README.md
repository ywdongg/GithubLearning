# Markdown 語法字典

## 標題語法
```markdown
# 一級標題
## 二級標題
### 三級標題
#### 四級標題
##### 五級標題
###### 六級標題
```

## 文字格式
- **粗體**: `**文字**`
- *斜體*: `*文字*`
- ~~刪除線~~: `~~文字~~`
- `程式碼`: ````程式碼```

## 列表
### 無序列表
```markdown
- 項目1
- 項目2
  - 子項目2.1
  - 子項目2.2
```

### 有序列表
```markdown
1. 第一項
2. 第二項
   1. 子項目2.1
   2. 子項目2.2
```

## 連結和圖片
- 連結: `[顯示文字](URL)`
- 圖片: `![替代文字](圖片URL)`

## 表格
```markdown
| 標題1 | 標題2 |
|-------|-------|
| 內容1 | 內容2 |
| 內容3 | 內容4 |
```

## 引用
```markdown
> 這是一段引用文字
>> 這是嵌套引用
```

## 程式碼區塊
````markdown
```python
def hello_world():
    print("Hello, World!")
```
````

## 分隔線
```markdown
---
或
***
```

## 待辦清單
```markdown
- [x] 已完成項目
- [ ] 未完成項目
```
